# AperoSplit - Google Play Store

## Titre
AperoSplit - Groupe

## Description courte
Partagez facilement vos dépenses de groupe

## Description complète
AperoSplit est l'application idéale pour gérer les dépenses de groupe entre amis, colocataires ou en voyage.

Fonctionnalités :
• Ajoutez et suivez les dépenses du groupe
• Visualisez les balances et qui doit quoi
• Simplifiez les dettes avec des suggestions de remboursement
• Statistiques et graphiques détaillés
• Notifications pour les paiements en attente
• Export CSV des dépenses
• QR Code pour inviter des membres
• Thème personnalisable avec 15 palettes de couleurs
• Authentification biométrique (Face ID / Empreinte digitale)
• Mode sombre disponible
• Fonctionne en français

Gérez facilement les dépenses de vos soirées, voyages, coloclocations ou tout autre groupe !
